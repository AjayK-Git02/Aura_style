# Aura Style 💎

![Aura Banner](assets/icons/app_logo.jpg)

**Aura Style** is a premium, AI-powered fashion assistant and virtual try-on application built with Flutter. It helps users discover their unique style personality ("Aura") and virtually try on outfits with photorealistic accuracy.

## ✨ Key Features

*   **🔮 Style Aura Quiz**: A scenario-based personality quiz that determines your fashion archetype (e.g., "The Architect", "The Mystic").
*   **👗 Virtual Try-On**: Advanced AI-powered virtual fitting room to see how clothes look on your avatar.
*   **🪄 Smart Wardrobe**: Digitize your closet with automatic background removal and categorization.
*   **👤 Design Persona**: Custom profile with style preferences, gender, and zodiac integration.
*   **🎨 Premium Experience**: A bespoke "Midnight Gold" dark theme with cinematic animations and luxury typography.

## 🛠️ Tech Stack

*   **Framework**: Flutter (Dart)
*   **Backend**: Supabase (Auth, Database, Storage)
*   **State Management**: Riverpod
*   **Routing**: GoRouter
*   **AI/ML**: Custom Background Removal & Style Analysis

## 🚀 Getting Started

1.  **Clone the repository**
    ```bash
    git clone https://github.com/AjayK-Git02/Aura_style.git
    ```

2.  **Install Dependencies**
    ```bash
    flutter pub get
    ```

3.  **Run the App**
    ```bash
    flutter run
    ```
